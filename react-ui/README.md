# Rock Paper Scissors React App

A simple, modern React app for playing Rock, Paper, Scissors against the computer.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Open your browser at [http://localhost:3000](http://localhost:3000)

## File Structure

- `src/RockPaperScissors.js` — Main game component
- `src/RockPaperScissors.css` — Game styles
- `src/index.js` — App entry point
- `public/index.html` — HTML template

## Features
- Play against the computer
- Score and round tracking
- Responsive, modern UI
- Reset button to restart the game

---

You can further customize the UI or add multiplayer features as needed.
